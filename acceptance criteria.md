Milestone 1 — Menu Bar App Shell

Done when:
	•	App appears in macOS menu bar
	•	Menu bar icon is visible
	•	Clicking menu bar icon opens a popover that loads local HTML file
	•	App is hidden from Dock (menu bar only)
	•	App runs via npm start

⸻

Milestone 2 — Static UI Replication

Done when:
	•	Menu bar icon shows timer status (e.g., time remaining)
	•	Popover opens when clicking menu bar icon
	•	Circular timer is visible in popover
	•	Time text is centered and legible
	•	Play/pause/reset controls are present in popover
	•	UI matches PRD layout (no logic yet)

⸻

Milestone 3 — Timer Engine

Done when:
	•	Focus timer counts down correctly
	•	Break timer counts down correctly
	•	Timers do not auto-start each other
	•	Completion triggers state change only

⸻

Milestone 4 — Pause, Reset & Custom Durations

Done when:
	•	Focus and break timers can be paused
	•	Timers can be reset at any time
	•	User can change durations
	•	Changes persist after restart

⸻

Milestone 5 — Break Task Database

Done when:
	•	User can add/edit/remove break tasks
	•	Tasks persist locally
	•	No daily logic yet

⸻

Milestone 6 — Daily Task Selection

Done when:
	•	User can select today's break tasks from database
	•	Only selected tasks appear during breaks
	•	Selection resets at midnight

⸻

Milestone 7 — Daily Tracking & Persistence

Done when:
	•	Focus sessions increment correctly
	•	Break task completion is tracked
	•	"Today X / Y" updates correctly
	•	All resets happen automatically by date

⸻

Milestone 8 — Packaging

Done when:
	•	App builds into a .app file
	•	Runs without terminal
	•	Works fully offline
